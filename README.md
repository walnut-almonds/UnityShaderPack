# Unity Shader Pack

A collection of custom Unity shaders (Built-in Render Pipeline) and an Editor tool to auto-generate example materials.

## Shaders Included
- Sand (procedural grain)
- Metal (PBR)
- Plastic
- Emissive
- Glass (refraction, IOR)
- Gemstone (IOR, dispersion)

## Usage
1. Import this folder/package into your Unity project (Built-in pipeline).
2. In Unity Editor, use `Tools > Create Example Materials` to auto-generate sample materials in `Assets/Materials`.
3. Assign materials to objects in your scene.

## Requirements
- Unity 2019.4+ (Built-in Render Pipeline)
